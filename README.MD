# TailwindCSS Coffee Style by NTNDev

## Requirements

- **nodejs >= v14.7.0**
- **npm >= v6.14.7**
- **yarn >= v1.19.1**

Clone project and run test.

```
$ git clone https://git_url_clone <project_dir>
$ cd <project_dir>
$ npm run dev
$ npm start
```

Demo: https://tailwind-coffee-ntndev.web.app/

Thanks for watching!
